Troubleshooting

  • If USB hardware not detected Try:
    • Unlplug ExoKey. Wait 3 seconds. Then plug-in ExoKey. Wait 15 seconds.
    • Reboot Windows with ExoKey Unplugged. After windows starts, plugin ExoKey, then start App
    • Look in windows device manager for ExoKey as a network device.
  • If Internet connection sharing fails.
    • Reboot Windows with ExoKey Unplugged. After windows starts, plugin ExoKey, then start App
    • Try to manually enable Control Panel\Network and Internet\Network Connections
      • Right click on the Internet facing WiFi or wired connection. Click Properties
      • Click on "Sharing" tab
      • Share internet to the exokey device
